Harry Potter and the Magical Class

August 10, 2015

Okay, past self...let's talk about wizards. And classes. But mostly wizards. Well, wizards as classes. Or wizards as class instances. Don't worry, we'll get to that. Starting now. 

        class Wizard

          def initialize(name)
            @name = name
          end

          def lumos
            p @name + " lights up the room."
          end

        end

Woah, did you see that? We basically just brought things (people) called Wizards into existence by creating a Wizard class! What, you missed it? Okay, so this part? 

        class Wizard
          ...
        end

That the creation of a "real-world object" that we've named Wizard. In between those lines? That's where the magic happens. Everything that starts with def ("define") is a method. It's something all wizards can do. But we have to define their abilities for them. Let's look at the first method. 

          def initialize(name)
            @name = name
          end

The initialize method is one that all classes must have. This will run as soon as a new wizard is created. Once a wizard is born, it has to have a name. Aside from it being tradition for a child to have a name at birth, we know that it's required because the initialize method expects a (name) parameter. The @ symbol makes the wizard's name accessbile from any other method in the Wizard class. It's called an instance variable. 

          def lumos
            p @name + " lights up the room!"
          end

The lumos method! As you can see, it uses the instance variable @name. And when it's "called" (when we ask the wizard to use lumos), the wizards lights up the room, and the statement in the middle prints out! Let's see it in action! 

        the_boy_who_lived = Wizard.new("Harry Potter")
        the_boy_who_lived.lumos

        => Harry Potter lights up the room!

What! You saw that, right? A wizard--The Boy Who Lived--is born. Straight away, he's given a name: Harry Potter. When we ask him to use the lumos method we defined in the Wizard class, he does! Excellent, Harry! Well done!

You know so much more now, past self! Just be careful with this new magic-like knowledge. As Professor Dumbledore once said, "The truth is a beautiful and terrible thing, and should therefore be treated with caution."
