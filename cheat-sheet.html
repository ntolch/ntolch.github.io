Which is easier?

It depends. Ruby's syntax, it's often argued, is more intuitive and simple. JavaScript, on the other hand, has syntax that, for looping, can be more self-explanatory.

While Loops

Ruby: 

        awesome_sauce = []
        i = 1
        while i < 4
          awesome_sauce.push("Cool thing #{i}")
          i += 1
        end

        =>awesome_sauce = ["Cool thing 1", "Cool thing 2", "Cool thing 3"]

JavaScript: 

        var awesome_sauce = [];
        i = 1;
        while (i < 4) {
          awesome_sauce.push("Cool thing " + i);
          i += 1;
        }

        =>awesome_sauce = ["Cool thing 1", "Cool thing 2", "Cool thing 3"]

Pretty similar. But JavaScript knows that this sort of while looping is pretty common, so it has its own shortcut for it--the for loop.

JavaScript for Loops: 

        for (var i = 1; i < 4; i++)
          awesome_sauce.push("Cool thing " + i);

For this for loop, the syntax goes like this: for (set counter variable; condition (the 'while' part of the loop); increment counter variable)
